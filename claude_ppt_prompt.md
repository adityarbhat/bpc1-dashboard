## Additional Instructions for PowerPoint Generation

**Consistency:**
- Use the same fonts throughout (Montserrat or similar)
- Maintain consistent spacing and margins
- Apply purple/magenta gradient to title slides and section breaks
- Use white backgrounds for content slides with dark purple text

**Transitions:**
- Use simple fade transitions between slides
- Avoid flashy animations (professional audience)
- Keep transitions under 0.5 seconds

**Image Placeholders:**
- Where specific images are needed (flowcharts, screenshots), include placeholder boxes with descriptive text
- Use icons from standard icon libraries
- Prioritize clarity over decoration

**Print-Friendly:**
- Ensure slides are readable if printed in black and white
- Use sufficient contrast for all text
- Avoid relying solely on color to convey information

**Accessibility:**
- Use minimum 18pt font for body text
- Use minimum 28pt font for titles
- Ensure adequate color contrast (WCAG AA compliant)
