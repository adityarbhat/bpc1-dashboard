# Module 1: Build a Mini Company Dashboard

## The Project

You'll build a simple company ratio viewer that connects all Week 1 concepts:

```
┌────────────────────────────────────────┐
│  Mini Company Dashboard                │
├────────────────────────────────────────┤
│                                        │
│  Select Company: [Ace ▼]               │  ← Exercise 2: Session State
│                                        │
│  ┌──────────────────────────────┐      │
│  │     Current Ratio: 2.15      │      │  ← Exercise 3: Gauge Chart
│  │         [Gauge]              │      │
│  └──────────────────────────────┘      │
│                                        │
│  Data fetched from Airtable            │  ← Exercise 1: Airtable
│  Last refreshed: 10:30 AM              │
│                                        │
└────────────────────────────────────────┘
```

## Exercise Flow

| Exercise | You Build | Uses From Previous |
|----------|-----------|-------------------|
| **01** | Fetch company ratios from Airtable | - |
| **02** | Add company selector with session state | Data from Ex 1 |
| **03** | Display ratio as gauge chart | Selector from Ex 2, Data from Ex 1 |
